🎯 四種選股方法

月頻

📈 revenue — 營收爆發

找出本月營收年增率突破 +20%，且相較上月加速的公司，確保動能是在加速而非減速。

🎯 核心條件 本月年增率 ≥ 20% 本月年增率 ≥ 上月年增率（加速） 📦 資料來源 MOPS 月報 → sql/income.db ⏰ 觸發 每月 10 日後 📊 排序 總分（由高到低）
日頻

🕯 pointK — K棒強度

當日紅K棒強度位於近 122 交易日（半年）前10強，捕捉主力異常買盤訊號。

🎯 核心條件 red_K_ratio ≥ 近122日第10名門檻 有效資料 ≥ 90 天 📦 資料來源 sql/stock_data.db ⏰ 觸發 每個交易日收盤後 📊 排序 總分（由高到低）
月頻

🏦 director — 董監增持

找出近一個月全體董監事淨買進超過 10 萬張的公司，追蹤內部人的真實行動。

🎯 核心條件 本月 − 上月持股 > 100,000 張 📦 資料來源 MOPS / GoodInfo ⏰ 觸發 每月持股公告後 📊 排序 總分（由高到低）
日頻

💼 fund — 投信買超

找出當日投信淨買超，以「投本比」衡量相對買力，突破大股本的干擾。市值無上限。

🎯 核心條件 投信買賣超股數 > 0 📐 投本比 買超股數 / (股本 × 100,000) 📦 資料來源 三大法人日報 ⏰ 觸發 每個交易日收盤後 📊 排序 投本比（由高到低）

⚖️ 九項評分系統

每支股票由以下 9 項指標加總得到總分，分數越高代表越值得買進。

① psrS / PSR 三年最低
當前預估PSR相對歷史低點的偏離程度
-2 ~ +2
# x = psrS / psr三年最低 - 1
y = clamp((0.2 - x) * 2, -2, +2)
-2（比三年低貴120%）+2（比三年低便宜20%）
② 股價淨值比
股價相對帳面價值的溢價程度
-2 ~ +2
if x >= 3.3:   y = -2
elif x >= 1.3: y = 1.3 - x
else:          y = (1.3 - x) * 2  # 折價加倍
淨值比分數
< 1.0接近 +2
1.30（零分線）
≥ 3.3-2 封頂
③ psrS 絕對值 ⭐
市值 / 預估今年營收，權重最高指標
-4 ~ +4
if psrs >= 3: y = -4
else: y = (1.5 - psrs) / 0.375
PSR分數
0.75 甜蜜點+2
1.5 零分線0
≥ 3-4 封頂
④ PRR — 市值/研發費用
PRR 越小代表研發護城河越深
-2 ~ +2
# prr = 市值 / 近4季研發費用
if prr >= 15 or prr <= 0: y = -2
else: y = (15 - prr) / 7.5
⑤ 毛利率
產品競爭力與定價能力
-2 ~ +2
if x < 10:   y = -2     # 代工廠
elif x >= 60: y = +2     # 品牌/軟體
else: y = (x - 30) / 15  # 30%為零分線
⑥ 營利率年增（YoY）
獲利能力是否在改善，看趨勢非絕對值
-2 ~ +2
# 本季營利率 - 前4季（同季YoY）
x = 本季營利率 - 前4季營利率
y = clamp(x / 2.5, -2, +2)
年增 +5% → 滿分，年增 -5% → 最低分
⑦ 營收年增 20%
月營收年增動能 + 加速度
-2 ~ +2
x = 最新單月年增率 / 100
w = 本年累計年增率 - 去年年增率  # 加速度
y = (x*100-10)*0.025 + w*0.25
前項：年增每超過10%基準40pp → +1分
後項：加速度每 +4% → +1分
⑧ 營收月增 80%
短期動能，捕捉爆發月
-2 ~ +2
x = 最新單月月增率 / 100
if x > 0.8:    y = +2
elif x < -0.8: y = -2
else:          y = x * 2.5
正常月份分數接近 0，主要抓極端爆發月
⑨ PEG
成長股是否被合理定價
-2 ~ +2
# 今年EPS預估（最新季加權×2）
今年 = 本季×2 + 前1季 + 前2季
去年 = 前4季×2 + 前5季 + 前6季
peg = 收盤 / 今年EPS / 成長率 / 100
PEG意義分數
< 0.66便宜成長股+2
1.0合理定價0
> 1.34高估-2

各指標滿分範圍總覽

#指標範圍分類核心邏輯
①psrS / PSR三年低-2 ~ +2估值相對歷史估值水準
②股價淨值比-2 ~ +2估值零分線 1.3x
③psrS 絕對值 ⭐-4 ~ +4估值甜蜜點 0.75x，唯一雙倍指標
④PRR 市值/研發-2 ~ +2估值研發護城河深度
⑤毛利率-2 ~ +2獲利零分線 30%
⑥營利率年增(YoY)-2 ~ +2獲利改善趨勢
⑦營收年增 20%-2 ~ +2動能年增率 + 加速度
⑧營收月增 80%-2 ~ +2動能爆發月捕捉
⑨PEG-2 ~ +2獲利成長定價合理性

💡 評分系統設計哲學

三層篩選確保每支入選股票同時具備「便宜」＋「品質」＋「動能」。

第一層：估值
-8 ~ +8 分
  • psrS 絕對值（±4）
  • psrS / PSR三年低（±2）
  • 股價淨值比（±2）
  • PRR 研發比（±2）
→ 確保安全邊際
第二層：獲利品質
-6 ~ +6 分
  • 毛利率（±2）
  • 營利率年增（±2）
  • PEG（±2）
→ 排除低價陷阱
第三層：動能催化
-4 ~ +4 分
  • 營收年增（±2）
  • 營收月增（±2）
→ 確認近期動能持續

理想得高分的股票輪廓

估值安全邊際PSR 低於歷史三年低點
獲利品質毛利率 > 30%，營利率持續改善
動能存在最新月營收年增 > 20% 且加速
成長被低估PEG < 1，市場尚未完全反映
籌碼配合董監或投信同步增持
