# 台股選股系統 快速開始指南 (Quick Start Guide)

本文件引導您從零開始快速建立環境、啟動系統並載入初始資料。

---

## 🚀 快速上手三步驟

### 步驟 1：建立虛擬環境與安裝套件
請在本專案的根目錄開啟 **PowerShell** 視窗，並執行以下指令來自動初始化環境與安裝所需套件：

```powershell
powershell -ExecutionPolicy Bypass -File .\setup_venv.ps1
```

* **此步驟會自動進行：**
  1. 偵測系統 Python 版本。
  2. 在目錄下建立 `venv` 虛擬環境。
  3. 升級 `pip` 並下載安裝 `requirements.txt` 中列出的所有核心依賴套件。
  4. 驗證所有模組是否皆可正常載入。

---

### 步驟 2：啟動網頁主控台
環境安裝成功後，直接雙擊專案目錄下的 `start_web.cmd` 檔案，或者在 PowerShell 中執行：

```powershell
.\start_web.cmd
```

* 執行後會自動打開瀏覽器，導向網頁主控台：**[http://127.0.0.1:5000/online/](http://127.0.0.1:5000/online/)**
* 💡 *請注意：背後的黑色命令列視窗為伺服器，**不可關閉**，關閉後網頁即會失效。*

---

### 步驟 3：網頁起來第一步 —— 執行「下載快照」取回資料
開啟網頁主控台後，您會看到系統資料庫均為「檔案不存在」或「沒有資料」的狀態。

請跟隨以下步驟進行初始化：
1. 在網頁主控台右側的 **【資料更新控制台】** 中，找到 **【3. 資料庫快照備份】** 區塊。
2. 點擊 **`下載快照`** 按鈕。
3. 系統將會自動背景下載最新版歷史數據壓縮檔 `gg_stock_data_XXXXXX.tar.gz`，並解壓更新至 `sql/stock_data.db` 資料庫與 `data/stock_data/` 個股 CSV 目錄。
   
   > [!IMPORTANT]
   > **關於 Google Drive 分享權限設定**  
   > 為了讓程式能夠在任何乾淨的環境下成功下載資料，請確保您在 Google Drive 上該檔案的分享權限已設定為 **「知道連結的任何人」(Anyone with the link) 均可檢視**。如果權限為「已限制」(Restricted)，程式在下載時會被重新導向至登入頁面並回報 401 錯誤。
   
4. 下載進度可在網頁左側的 **【實時控制台日誌】** 視窗中即時查看。當日誌顯示「Download completed successfully!」且任務狀態變回「空閒」時，代表初始化成功！

---

## 📅 日常盤後操作說明

資料快照載入完畢後，您就可以開始每日的盤後選股操作：

1. **抓取當日新資料**：
   * 在主控台分別執行 **`抓取個股日K`**、**`抓取三大法人`**、**`抓取本益比淨值比`**。
2. **執行選股**：
   * 點擊右下角的 **`執行選股`**，即可產出最新的 HTML 報表。
   * 您可以直接點擊主控台底部的 HTML 連結（例如 `籌碼+基本面精選好股 (Chip & Fund)`）在瀏覽器中查看精美的動態篩選報表。
